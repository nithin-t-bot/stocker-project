# 📈 Stocker – Cloud Based Stock Trading Application

Stocker is a **cloud-based stock trading simulation platform** that allows users to practice buying and selling stocks without risking real money.
The application helps beginners understand stock market concepts and analyze trading strategies in a safe environment.

---

## 🚀 Features

* 🔐 Secure user authentication
* 📊 Real-time stock price visualization
* 💰 Simulated buy and sell trading
* 📁 Portfolio tracking
* 📈 Stock market analysis
* 🌐 User-friendly web interface

---

## ⚙ Installation

### 1️⃣ Clone the repository

```
git clone https://github.com/nithin-t-bot/stocker-project.git
```

### 2️⃣ Navigate to project folder

```
cd stocker-project
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the application

```
python app.py
```

---

## 🎯 Problem Statement

Many beginners want to learn stock trading but do not have a **safe environment to practice trading strategies**.

Stocker solves this by providing a **risk-free simulated trading platform** where users can explore stock trading concepts and track their performance.
